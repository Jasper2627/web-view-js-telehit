Este pequeño programa solo muestra la composición de un plugin sencillo hecho en js, que consume feeds para actualizarse
automáticamente.

En algunas secciones no habrán imágenes visibles ya que está configurado para verse en un servidor local de tipo localhost, 
sin embargo en otras secciones si se mostrarán ya que recurre a una peticción directamente de la web que es donde se
alimenta la aplicación estando en producción.

Por otro lado no existe un menú de navegación, las secciones podrán verlas si se envía el valor de la variable "val" vía get.

Próximamente actualizaré la aplicación para un funcionamiento adecuado, sin embargo esta, es la base. Se agradecen sus
contribuciones.
